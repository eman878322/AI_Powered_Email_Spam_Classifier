# AI-Powered Email Spam Classifier 🚀

An end-to-end Machine Learning and Natural Language Processing (NLP) pipeline that automatically detects and classifies emails into **Spam** or **Ham** (Legitimate).

## 📊 Project Overview
This project focuses on filtering out malicious and unwanted emails using advanced text preprocessing and machine learning classification algorithms. 

## 🛠️ Features & Technical Pipeline
- **Text Preprocessing:** Lowercase conversion, punctuation/number removal via Regex, and English stopwords filtering.
- **Feature Extraction:** Word vectorization using **TF-IDF Vectorizer** (Top 3,000 features).
- **Model Comparison:** - **Naive Bayes:** 94.59% Accuracy
  - **Logistic Regression:** 97.87% Accuracy 🔥
- **Evaluation:** Detailed classification report, confusion matrix visualization (Heatmap), and a live custom message testing function.
